Files:
- sequence_diagram.png: sequence diagram of the assignment
- class_diagram.png: class diagram of the assignment
- src/*: source codes of the assignment
- .classpath: class path of the project
- .project: configs of the project
- README.txt: instructions of files and setup

Setup:
- Add the project into Eclipse
- In Package Explorer, right-click on both Host.java, choose Run As -> 1 Java Application
- Still in Package Explorer, right-click on Client.java and Server.java, and choose Run As -> 1 Java Application
- Check output in the console
